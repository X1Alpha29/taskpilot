# 🚀 TaskPilot

A full-stack SaaS task management application with AI-powered features.

---

## ✨ Features

- 🔐 Authentication (Register / Login with JWT)
- 📁 Project management
- ✅ Task creation, editing, drag & drop (Kanban style)
- 🧠 AI-powered:
  - Task generation
  - Project insights
- 📊 Analytics dashboard
- 💬 Comments system
- 📎 File uploads
- 🧑 Account deletion

---

## 🛠 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- DnD Kit (drag & drop)

### Backend
- Flask
- SQLAlchemy
- JWT Authentication

### AI Integration
- OpenAI API
